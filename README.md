# FinChain Sentinel

An end-to-end data engineering project that detects financial anomalies
in supply chain transactions using AWS, Airflow, Snowflake, and Python.

## Tech Stack
- Ingestion: Python, AWS S3, AWS Lambda
- Orchestration: Apache Airflow
- Transformation: PySpark, AWS Glue, dbt
- Storage: Snowflake, AWS Redshift
- Detection: scikit-learn (Isolation Forest)
- Visualization: Power BI
